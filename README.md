# NTP-clock-TFT
A simple clock with a TFT display, obtains time from NTP via the Internet.

## Use

1. ESP32C3 micro
2. Waveshare 1.9 inch LCD Module 170x320 pixels

## Connection:

ESP32 – LCD<br>
GND - GND<br>
3V3 - VCC<br>
3V3 - BL<br>
GPIO3 – RST<br>
GPIO2 – DC<br>
GPIO7 – CS<br>
GPIO4 - CLK<br>
GPIO6 - DIN<br>

<i>This program was designed for the Arduino IDE C++.</i>
